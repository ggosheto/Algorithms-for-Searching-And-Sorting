# Algorithm Playground

A collection of fundamental **Searching** and **Sorting** algorithms implemented in C#. This repository serves as a reference for understanding algorithmic logic, time complexity, and efficient data manipulation.

---

## 🔍 Searching Algorithms

These algorithms are used to find the location of a specific element within a data structure.

* **Linear Search:** A straightforward approach that checks every element sequentially.
* **Binary Search:** An efficient $O(\log n)$ algorithm for finding elements in a **sorted** array by repeatedly dividing the search interval in half.

---

## 📊 Sorting Algorithms

A variety of comparison-based and non-comparison-based sorting techniques.

### Comparison Sorts

* **Bubble Sort:** A simple algorithm that repeatedly steps through the list and swaps adjacent elements.
* **Selection Sort:** Maintains a sorted and unsorted boundary, picking the smallest element from the unsorted part each time.
* **Insertion Sort:** Builds the final sorted array one item at a time by "inserting" elements into their correct position.
* **Quick Sort:** A divide-and-conquer algorithm that picks a 'pivot' and partitions the array around it.
* **Merge Sort:** A recursive divide-and-conquer approach that splits the array in half, sorts them, and merges them back together.

### Non-Comparison Sorts

* **Counting Sort:** An integer sorting algorithm that operates by counting the number of objects that have each distinct key value.
* **Radix Sort:** A non-comparative integer sorting algorithm that sorts data with integer keys by grouping keys by the individual digits which share the same significant position and value.
